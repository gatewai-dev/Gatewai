This section provides detailed documentation for all the nodes available in Gatewai. Nodes are the building blocks of your workflows, allowing you to process data, generate content, and create complex logic.

## Categories

Nodes are organized into the following categories:

- **Inputs**: Nodes that provide data to the graph (e.g., Text, File, Number).
- **Outputs**: Nodes that display or export results (e.g., Preview, Export).
- **AI**: Generative AI nodes for images, video, text, and audio.
- **Image**: Image processing nodes (e.g., Crop, Resize, Blur).
- **Video**: Video processing nodes.
- **Tools**: Utility nodes (e.g., Text Merger, Sticky Note).
