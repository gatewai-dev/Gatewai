**Category**: Image

Adjusts hue, saturation, lightness, and brightness.

## Inputs
- **Image** (Image): The source image.

## Outputs
- **Result** (Image): The modulated image.

## Configuration
- **Hue**: 0
- **Saturation**: 1
- **Lightness**: 1
- **Brightness**: 1
