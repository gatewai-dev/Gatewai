---
title: Paint
description: Draw / Fill Mask on an image.
---

**Category**: Image

Allows drawing on an image or creating a mask.

## Inputs
- **Background Image** (Image): Optional background.

## Outputs
- **Image** (Image): The resulting image with drawings.
- **Mask** (Image): The binary mask of the drawing.

## Configuration
- **Width**: 1024
- **Height**: 1024
- **Maintain Aspect**: true
- **Background Color**: #000
