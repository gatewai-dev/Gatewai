## Why Yodes?

Yodes is a powerful platform designed to bridge the gap between complex generative AI models and usable, visual workflows.

<CardGrid stagger>
	<Card title="For Users" icon="pencil">
		**Real-time Previews**: See your changes instantly directly on the canvas.
        **Visual Workflow Builder**: Intuitive node-based interface to chain AI models and image processing tools.
        **Hybrid Execution**: Seamlessly offload heavy tasks to the server while keeping UI interactions snappy.
	</Card>
	<Card title="For Developers" icon="laptop">
		**API Integration**: Turn any canvas into an API endpoint.
        **Extensible Architecture**: Built on a modern stack (React, Hono, Prisma) designed for scalability.
        **Custom Nodes**: Easily extend functionality by creating new node templates.
	</Card>
</CardGrid>

## Key Features

- **Multi-Modal Support**: Work with Text, Image, Video, and Audio in a single graph.
- **Smart Batching**: Efficient execution of complex dependency graphs.
- **Asset Management**: Built-in handling for uploaded and generated media files.
